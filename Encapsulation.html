Encapsulation

Define:

encapsulation is a protective shield that prevents the data from being accessed by the code outside this shield the variables or data of a class are hidden from any other class and can be accessed only through any member function of its own class in which they are declared



When to define a variable:

           
     
            public class Person
            {
            private string _name;  
            }
get and set 

           
     
            public class Person

            private string _name;

            }
